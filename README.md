# Fetch

```
git clone git@github.com:thydel/gmk.git
```

# Install

```
make -C gmk -f gmk install
```

# Fill gmk/.git/info/exclude

```
gmk -C gmk exclude
```

# Set up local git config

```
gmk -C gmk self/config
```
